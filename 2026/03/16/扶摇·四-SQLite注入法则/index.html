SQLite 注入基础与 MySQL 的差异

SQLite 作为一款轻量级的嵌入式数据库，在移动应用和小型 Web 项目中广泛应用。与 MySQL 相比，SQLite 注入存在几个关键差异点：

**系统函数差异：**SQLite 缺少 MySQL 中常用的 mid() 、 left() 、 sleep() 甚至 if() 等函数

注释符号 ：SQLite 使用 – （注意空格）或 /* */ 作为注释，而非 MySQL 的 #

元数据查询 ：SQLite 没有 information_schema ，而是使用 sqlite_master 表存储元数据

字符串连接 ：使用 || 而非 MySQL 的 concat()
